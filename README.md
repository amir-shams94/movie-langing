# Movie-landingPage
this project setup with Vue 3 composition api and pina 
and ui setup with simple Bulma
## dependency:
Bulma
Pinia
Vue Router
Axios 

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
